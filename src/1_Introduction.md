# Introduction

This book will cover the basics of creating plugins for the X-Plane flight simulator using the Rust language.

X-Plane developers provide a C API for writing plugins. The `xplm_oxide` crate is a Rust wrapper around this C API. It provides a safe and idiomatic Rust interface for writing X-Plane plugins.

## Who is thos book for?

This book is for anyone with some basic understanding of the Rust language who wants to write plugins for X-Plane. It is assumed that the reader has some basic knowledge of Rust, such as how to create a new project, how to add dependencies, and how to build and run a project.

If you do not have any experience with Rust, I recommend you use following resources first:

- [Rust Book](https://doc.rust-lang.org/book/). The official Rust book. It is a comprehensive resource for learning Rust.
- [Rustlings](https://rustlings.rust-lang.org/). A series of small exercises to get you used to reading and writing Rust code.
- [Rust By Example](https://doc.rust-lang.org/rust-by-example/). A collection of runnable examples that illustrate various Rust concepts and standard libraries. A great accompanying resource to the Rust Book.
- [Rust Documentation Site](https://doc.rust-lang.org/stable/). The official Rust documentation site. It provides detailed information about the Rust language and its standard library.
